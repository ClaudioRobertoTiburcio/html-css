Principais Formatações

Negrito / Destaque

Nessa frase temos um termo em negrito usando a tag B (Não semantica).

Nesta frase, tenho um termo em destaque usando a tag STRONG (semantica).

Itálico / Enfase

Nesta frase, temos um termo em itálico usando a tag I (Não semantica).

Nesta frase, temos um termo em enfase usando a tag EM (semantica).

Texto marcado

Podemos também criar um texto marcado usando a tag MARK.

E no outro parágrafo, temos temos outro texto marcado no final.

Texto grande e pequeno

É possível criar um texto grande(Porém, obsoleto) com a tag BIG e um texto pequeno com a tag SMALL nesse parágrafo.

Texto deletado

Podemos marcar um texto como excluído usando a tag DEL, para indicar que ele pode ser lido, mas deve ser desconsiderado.

Texto inserido

Podemos marcar um texto como inserido usando a tag INS, para dar uma ênfase e indicar que ele foi adicionado depois.

Texto sobrescrito

É usado para fazer o texto ser um pouco mais alto que o normal, como é o caso de um número elevado a outro, por exemplo: 37

Texto subescrito

Texto subescrito é usado para mostrar algo deslocado um pouco para baixo como em fórmulas quimícas. Ex.: H2O
